# Client Module Structure

## Main Directories

- **IO/** - Handles input/output operations.
- **Modules/** - Contains modular components for the client.
- **SignalR/** - Manages real-time communication using SignalR.

---

## Key Files

- **Client.fsproj** - Project file defining dependencies and build instructions.
- **Effect.fs** - Manages side effects within the client.
- **Misc.fs** - Miscellaneous utility functions used across the project.
- **Program.fs** - Entry point of the client application.
- **Runtime.fs** - Defines runtime behaviors and configurations.
- **Types.fs** - Type definitions used within the client module.
