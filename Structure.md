# Project Structure

## Main Directories

- **Client/** - Contains the client-side application code.
- **Client.Tests/** - Unit tests for the client-side code.
- **Common/** - Shared components and utilities across the project.
- **Server/** - Server-side application logic and API endpoints.

---

## Key Files

- **Jarvis.sln** - Solution file managing all project components.
- **LICENSE** - Licensing information for the project.
- **readme.md** - Project overview and documentation.
- **todo.md** - Pending tasks and development roadmap.
- **ProcessTracker.md** - Tracks project progress and workflow.
- **Structure.md** - Overview of the project folder structure. to further describe its contents when necessary.
